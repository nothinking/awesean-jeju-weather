# awesean-jeju-weather

제주도 실시간 날씨를 눈으로 확인하는 관제 대시보드.

제주는 지역별로 날씨가 천차만별이라, CCTV 실황과 기상 데이터를 한 화면에 모아 현재 날씨를 직관적으로 파악할 수 있도록 만들었습니다.

## 주요 기능

**CCTV 실시간 영상 (16개 카메라)**
- 북부: 제주공항, 용두암 해안, 탑동 해안
- 남부: 서귀포 항, 법환포구, 법환 해안, 중문 해안
- 동부: 성산일출봉, 온평 해안
- 서부: 신창 해안, 화순 해안
- 한라산: 백록담, 왕관릉, 윗세오름, 어승생악, 1100도로

**Windy.com 날씨 지도**
- 🛰 위성 영상 (Satellite)
- 🌧 강수량 (Rain)
- 💨 바람 (Wind)

**대시보드 기능**
- 통합 관제 / CCTV 전체 / 날씨 지도 탭 전환
- 지역별 필터링 (북부/남부/동부/서부/한라산)
- 그리드 레이아웃 변경 (2×2, 3×3, 4×4)
- 더블클릭으로 개별 카메라 전체화면 확대
- ESC로 전체화면 닫기

## 사용법

`index.html` 파일을 브라우저에서 열면 바로 사용 가능합니다. 별도 서버나 설치가 필요 없는 단일 HTML 파일입니다.
